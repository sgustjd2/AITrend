## 1. 모델 & API 동향

### GPT-Realtime-2 — OpenAI 최초 GPT-5급 실시간 음성 추론 모델

OpenAI가 **GPT-Realtime-2**를 포함한 세 가지 실시간 오디오 모델을 기습 공개했다. 이번 발표는 음성 AI를 단순 응답 도구에서 '실제 작업 수행 에이전트'로 격상시키는 전환점이다.

**GPT-Realtime-2 주요 특징:**
- OpenAI 최초의 **GPT-5급 추론 능력**을 갖춘 음성 모델
- 대화 중 말 끊김, 요청 변경에도 자연스럽게 문맥 유지
- **128K 토큰 컨텍스트 윈도우** 지원
- **Voice-to-action** — 캘린더 조회, 외부 앱 도구 조작 등 실제 작업 동시 수행
- 가격: 오디오 입력 $32/1M 토큰, 출력 $64/1M 토큰

**동반 공개 신모델:**

| 모델 | 기능 | 가격 |
|------|------|------|
| GPT-Realtime-Translate | 70개+ 입력 언어 → 13개 출력 언어 실시간 통역 | $0.034/분 |
| GPT-Realtime-Whisper | 말하는 동안 실시간 스트리밍 음성 → 텍스트 변환 | $0.017/분 |

> 포인트: '대화하면서 동시에 일 처리'하는 Voice-to-action 흐름이 실제 API로 제공되기 시작했다. 음성 AI가 UX 레이어를 넘어 백엔드 에이전트로 진화하는 시점

## 2. 헬스케어 AI

### Google Health + Fitbit 통합 — Gemini AI 코치와 24시간 건강 관리

구글이 **Fitbit을 완전히 Google Health로 흡수**한다. 5월 26일부터 Fitbit 앱은 안드로이드·iOS에서 "Google Health" 앱으로 교체된다.

**핵심 변화:**
- Health Connect, Apple Health, 의료 기록까지 **단일 플랫폼에서 통합 관리**
- **Gemini 기반 Google Health Coach** — 운동·수면·회복·웰빙 데이터 분석, 24시간 맞춤형 건강 조언
- **Fitbit Air** ($99) — 화면 없는 신형 트래커. AI 코치에게 끊임없이 정확한 생체 데이터를 공급하는 역할

> 포인트: 헬스 데이터의 파편화(여러 기기·앱)를 끝내고 'AI가 데이터를 통합 해석·코칭'하는 플랫폼 전쟁이 본격화됐다. 구글이 헬스케어 AI 플랫폼의 표준을 노린다

## 3. 인프라 & 훈련 플랫폼

### Prime Intellect Lab — 자가 학습 에이전트 훈련 플랫폼 정식 출시

**Prime Intellect**가 자가 학습 에이전트를 위한 훈련 플랫폼 **Lab**을 정식 출시했다.

**주요 특징:**
- 사용자 데이터 기반 AI 직접 훈련 + 강화학습(RL) 엔진 제공
- **Nvidia, OpenAI, Meta** 등 주요 모델 지원
- **prime-rl 기반 비동기 강화학습** — GPU 클러스터 관리 부담 없이 1B~400B 규모 모델 학습
- 베타 기간 이미 **1만 건 이상의 훈련 작업** 수행
- 기업이 실제 작업 피드백으로 스스로 성능을 높이는 **자기 개선형 시스템** 구축 가능

> 포인트: 모델을 쓰는 것에서 나아가, 자신만의 데이터로 모델을 직접 진화시키는 'AI 자가 개선 루프'가 일반 기업에게도 열렸다

## 4. 개발자 도구

### Chromex — Codex 기반 오픈소스 Chrome AI 사이드패널

Chrome에서 사용할 수 있는 AI 도구 **Chromex**가 공식 출시됐다.

**주요 특징:**
- **ChatGPT 계정만 있으면 즉시 사용** 가능
- Codex 기반으로 구동, Chrome 사이드패널에서 동작
- 페이지 컨텍스트 이해, 탭 관리, 음성, 이미지 워크플로우 지원
- **100% 오픈소스** (GitHub: GENEXIS-AI/chromex)

> 포인트: 브라우저 자체가 AI 에이전트 실행 환경이 되는 흐름이 가속화 중. 오픈소스로 공개되어 커스터마이징 및 자체 배포 활용도 가능
