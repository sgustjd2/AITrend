## 1. 모델 출시 & 업데이트

### Claude Code `/goal` 기능 정식 추가

Anthropic이 Claude Code에 작업 완료 조건을 설정하는 **`/goal`** 기능을 정식 추가했다. 사용자가 "모든 테스트 통과 + lint clean" 같은 목표를 설정하면 Claude가 조건을 충족할 때까지 자율적으로 반복 작업한다.

**핵심 특징:**

| 항목 | 내용 |
|------|------|
| 기능 | 사용자 정의 완료 조건 설정 |
| 동작 | 조건 충족까지 Claude가 지속 반복 실행 |
| 연관 기능 | `/loop`, `/schedule`, stop hook |
| 의미 | 단순 코드 생성기 → 지속 실행형 에이전트 운영 환경으로 진화 |

> 포인트: Codex의 조건 기반 에이전트 루프 개념을 Claude Code가 공식 흡수했다. 개발자가 원하는 결과를 명시하면 Claude가 알아서 도달하는 **목표 지향형 자율 코딩** 패러다임이 열린다.

### MiniCPM-V 4.6 — 최경량 산업 안전 모니터링 추천

MiniCPM-V 4.6는 SigLIP2-400M 인코더 + Qwen3.5-0.8B LLM 기반으로 **1.3B 파라미터**에 Qwen3.5-2B 수준의 시각 이해 성능을 달성했다.

**벤치마크 비교:**

| 모델 | AAII 점수 | 토큰 비용 | 비고 |
|------|---------|---------|------|
| MiniCPM-V 4.6 (1.3B) | 13 | 기준 | OpenCompass·RefCOCO·OCRBench 우위 |
| Qwen3.5-0.8B | 10 | 19배 높음 | |

```python
from transformers import AutoModel, AutoTokenizer
import torch

model = AutoModel.from_pretrained(
    "openbmb/MiniCPM-V-4.6",
    torch_dtype=torch.bfloat16,
    device_map="auto",
    trust_remote_code=True,
)

# 산업 안전 Vision Q&A 예시
questions = [
    "현재 이미지에 화재나 연기가 보이나요?",
    "작업자가 안전모와 안전고리를 착용하고 있나요?",
    "이 이미지에서 위험 요소가 있다면 설명해주세요.",
]
```

### LFM2.5-VL-450M — 초경량 엣지 VLM (Liquid AI)

Liquid AI의 **LFM2.5-VL-450M**은 450M 파라미터로 **250ms 이하 엣지 추론**을 달성한 초경량 VLM이다.

| 항목 | 내용 |
|------|------|
| 파라미터 | 450M |
| 훈련 토큰 | 28T (이전 LFM2 대비 10T→28T 확장) |
| 핵심 기능 | 바운딩 박스 예측, 멀티언어 시각 이해 (한국어 포함) |
| 지원 언어 | 아랍어, 중국어, 프랑스어, 독일어, 일본어, 한국어, 포르투갈어, 스페인어 |
| 배포 환경 | NVIDIA Jetson Orin, AMD Ryzen AI Max, Snapdragon 8 Elite |
| 포맷 | HuggingFace, GGUF, ONNX |

> 포인트: 450M이면 RTX 3070(8GB)에서 배치 추론이 매우 여유롭다. 바운딩 박스 예측 기능이 내장돼 화재·연기·안전장비 위치 검출에 바로 활용 가능하다.

---

### 화재·안전 감지 특화 모델 조합 추천

| 목적 | 추천 조합 | 이유 |
|------|---------|------|
| RTX 3070 (8GB), 빠른 응답 | LFM2.5-VL-450M | 초경량, 250ms 이하, 바운딩 박스 포함 |
| RTX 3070 (8GB), 높은 정확도 | MiniCPM-V 4.6 (1.3B) | 낮은 VRAM, 뛰어난 시각 이해 |
| RTX 4070 (12GB), 균형 | Qwen3-VL-4B | 멀티언어, 256K 컨텍스트, 강한 추론 |
| RTX 4070 (12GB), 화재 특화 | ForestFireVLM-7B | 산불·화재 설명 생성 특화 파인튜닝 |

> 포인트: 화재·연기 감지 + 안전장비 확인 용도라면 **LFM2.5-VL-450M으로 실시간 탐지 → MiniCPM-V 4.6로 상세 Q&A** 파이프라인이 RTX 3070에서 가장 현실적인 조합이다.
