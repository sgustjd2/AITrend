## 2. 경량 VLM (Vision Language Model) 업데이트

> 용도: 이미지 분석 (화재/연기 감지, 작업자 안전고리 착용 여부 확인 등)
> 타겟 GPU: RTX 3070 (8GB VRAM) / RTX 4070 (12GB VRAM)

---

### ✅ 추천 1순위: MiniCPM-V 4.6 (2026년 5월 11일 출시)

| 항목 | 내용 |
|------|------|
| 파라미터 | **1.3B** |
| 비전 인코더 | SigLIP2-400M |
| 언어 백본 | Qwen3.5-0.8B |
| 컨텍스트 | 262k 토큰 |
| 라이센스 | Apache 2.0 (오픈소스) |
| 지원 플랫폼 | vLLM, SGLang, llama.cpp, Ollama |

**RTX 3070/4070 호환성**: ✅ 완벽 호환 (1.3B는 4~6GB VRAM으로 구동 가능)

**핵심 특징**:
- LLaVA-UHD v4 기반 초효율 아키텍처
- 시각 인코딩 연산량 **50% 이상 감소**
- 단일 이미지, 다중 이미지, 비디오 이해 지원
- iOS/Android/HarmonyOS 모바일 배포 가능

**안전 점검 활용 예시**:
```python
from transformers import AutoModel, AutoTokenizer
model = AutoModel.from_pretrained('openbmb/MiniCPM-V-4_6', trust_remote_code=True)

# 화재/연기 감지
question = "현재 이미지에 화재나 연기가 있나요? 있다면 위치와 심각도를 알려주세요."

# 안전고리 착용 확인
question = "현재 작업자는 안전고리(안전벨트)를 착용하고 있나요? 착용 여부와 올바른 착용 방법을 확인해주세요."
```

### ✅ 추천 2순위: SmolVLM-2B (Hugging Face)

| VRAM 사용량 | **5.02GB** (가장 메모리 효율적) |

**RTX 3070/4070 호환성**: ✅ 완벽 호환 (5GB VRAM으로 구동)

**벤치마크**:
| 테스트 | 점수 |
|--------|------|
| DocVQA | 81.6 |
| TextVQA | 72.7 |
| MMStar | 42.1 |

**장점**: 
- Qwen2-VL 2B(13.7GB) 대비 VRAM 3배 절약
- InternVL2 2B(10.5GB) 대비 VRAM 2배 절약
