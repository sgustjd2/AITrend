### 신규 주목 — NVIDIA LocateAnything (HF Papers 90표)

**LocateAnything** — NVIDIA가 발표한 고속 시각-언어 그라운딩 모델. 텍스트 쿼리로 이미지 내 객체 위치를 병렬 박스 디코딩으로 빠르게 특정한다.

| 항목 | 내용 |
|------|------|
| 기관 | **NVIDIA Research** |
| 핵심 기술 | Parallel Box Decoding — 객체 위치를 병렬로 빠르게 디코딩 |
| 강점 | "화재 감지기 위치", "안전고리 버클 위치" 등 특정 객체 좌표 반환 |
| 안전 감지 활용 | 작업자 신체 특정 부위 안전 장비 착용 여부 픽셀 수준 확인 |
| 속도 | 기존 방법 대비 빠른 추론 속도 (실시간 CCTV 적용 가능) |

> 포인트: 기존 VLM이 "착용했나요?"를 예/아니오로 답한다면, LocateAnything은 "안전고리가 이미지의 이 좌표에 있다/없다"를 반환한다. 탐지 정밀도가 필요한 산업 안전 시스템에 직접 활용 가능한 아키텍처.

### 빠른 실행 — SmolVLM-256M (초경량 CCTV 실시간 감지)

```python
from transformers import AutoProcessor, AutoModelForVision2Seq
from PIL import Image
import torch

# 초경량 256M — RTX 3070 VRAM 1GB 이하
processor = AutoProcessor.from_pretrained("HuggingFaceTB/SmolVLM-256M-Instruct")
model = AutoModelForVision2Seq.from_pretrained(
    "HuggingFaceTB/SmolVLM-256M-Instruct",
    torch_dtype=torch.float16,
    device_map="auto"
)

image = Image.open("cctv_frame.jpg")
messages = [
    {
        "role": "user",
        "content": [
            {"type": "image"},
            {"type": "text", "text": "다음 항목을 각각 예/아니오로 답하세요:\n1. 화재 또는 연기가 보입니까?\n2. 작업자가 안전고리(하네스)를 착용하고 있습니까?\n3. 안전모를 착용하고 있습니까?"}
        ]
    }
]

prompt = processor.apply_chat_template(messages, add_generation_prompt=True)
inputs = processor(text=prompt, images=[image], return_tensors="pt").to(model.device)
outputs = model.generate(**inputs, max_new_tokens=150)
print(processor.decode(outputs[0][inputs["input_ids"].shape[-1]:], skip_special_tokens=True))
```

### 주목 — LLaVA-OneVision-2 (961표 압도적 1위)

**LLaVA-OneVision-2**는 차세대 멀티모달 퍼셉션 인텔리전스 모델로 HF 일간 논문 최상위 투표를 받았다.

> 포인트: LLaVA 시리즈는 경량·오픈소스 VLM의 대표 격이다. OneVision-2의 등장은 단일 이미지에서 동영상, 다중 이미지 이해까지 아우르는 통합 비전 능력의 새 기준을 제시한다. RTX 3070/4070 구동 여부는 공식 파라미터 확인 후 업데이트 예정.

### 주목 — LocateAnything (NVIDIA, 90표)

**LocateAnything**은 텍스트로 "무엇을 찾아라"고 지시하면 이미지 내 해당 객체의 정확한 위치(바운딩 박스)를 고속으로 반환하는 모델이다.

> 포인트: 안전 감시 시스템에서 "작업자 중 안전고리 미착용자의 위치를 모두 표시하라"는 지시를 실시간 CCTV 프레임에 적용 가능하다. YOLO 계열 탐지 모델과의 조합보다 자연어 지시가 직관적이라 현장 맞춤 활용이 쉽다.

### 주목 — MobileGMoE (Meta AI)

**MobileGMoE**는 온디바이스(스마트폰·엣지 디바이스)에서 Mixture of Experts(MoE) 구조를 구현하는 연구다.

> 포인트: 엣지 디바이스에서 MoE를 돌린다는 것은 작은 모델이지만 전문화된 '전문가' 다수를 상황에 따라 선택 호출하는 구조로 실행됨을 의미한다. CCTV 카메라 자체나 엣지 서버에서 경량 VLM을 실행하는 미래 아키텍처의 선행 연구.
