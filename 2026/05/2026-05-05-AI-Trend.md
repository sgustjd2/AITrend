## 1. 모델 & 제품 동향

### GPT-5.5 Instant — 모든 ChatGPT 사용자 기본 모델 전환

OpenAI가 **GPT-5.5 Instant**를 새롭게 공개하고 ChatGPT 전체 사용자의 기본 모델로 적용했다.

**주요 변화:**
- '간결함'과 '개인화'를 핵심 방향으로 설정
- 기계적·장황한 설명 대신 따뜻하고 자연스러운 어투로 핵심만 전달
- 과거 대화 기록, 파일, 연결된 Gmail 맥락까지 기억해 맞춤형 답변 생성

> 포인트: 모델 지능이 평준화되면서 경쟁의 무게중심이 '기술력'에서 '대화 센스와 개인화'로 이동 중

**GPT 5.3 vs GPT 5.5 비교 내용도 함께 공개**

---

### Gemini 3.2 Flash — 구글 I/O 전 유출

iOS 앱에서 **Gemini 3.2 Flash** 모델이 포착되며 주목받고 있다.

**주요 특징:**
- 넘버링이 3.5가 아닌 3.2 → 완전한 새 패러다임보다 성능 최적화·안정성 집중 의도로 해석
- 복잡한 ASCII 애니메이션을 일관되게 생성, 가장 가벼운 체급임에도 기존 3.1 Pro 수준 성능 평가
- **Gemini Agent UI** 동시 유출 — 스케줄·작업 관리 전용 인터페이스

> 포인트: 구글 I/O에서 공개될 큰 그림의 일부가 사전 노출된 것으로 보임

---

### Google AI Studio + NanoBanana — 이미지 생성 통합

구글 AI 스튜디오에 이미지 생성 모델 **NanoBanana**가 새롭게 통합되었다.

- 프롬프트로 앱을 만들면 기획에 맞는 맞춤형 이미지 에셋을 실시간 자동 생성
- 새로 디자인된 시각적 편집 도구로 코드 없이 컴포넌트 수정·이미지 교체 가능

## 2. Anthropic 업데이트

### Claude 금융 특화 에이전트 템플릿 공개

Anthropic이 **기업 가치 평가, 피칭 자료 작성, 월말 결산** 등 금융 실무에 특화된 에이전트 템플릿을 공개했다.

**핵심 내용:**
- **FactSet**, **S&P 글로벌** 같은 전문 금융 데이터와 직접 연동
- 에이전트 하나에 필요한 커넥터 + 하위 에이전트들이 패키지로 묶여 제공
- **Claude Cowork** 또는 **Claude Code**에 플러그인 형태로 즉시 설치·사용 가능
- Excel, PowerPoint 실무 워크플로우 직접 지원

> 포인트: 금융권 월말 결산 야근을 AI가 대체하는 시나리오가 현실화되고 있음

---

### Anthropic 키리스 인증(Keyless Auth) 도입

Claude 플랫폼에 **키리스 인증** 기능이 추가되었다.

- API 키 발급·관리 없이 **AWS, GCP, Azure** 기존 클라우드 권한(IAM) 또는 **OIDC 토큰**만으로 Claude 접근 가능
- 주기적 키 교체·관리 부담 해소
- GitHub에 실수로 API 키를 노출하는 보안 사고 위험 원천 차단

---

## 3. 플랫폼 & 생태계 확장

### Ollama + Claude Desktop 공식 연동

**Ollama**가 **Claude Desktop(Claude Cowork)** 과의 연동을 공식 발표했다.

**의미:**
- 명령어 한 줄로 Anthropic 모델 대신 Ollama의 로컬/클라우드 모델(예: Kimi K2.6)을 Claude Cowork·Code에서 구동 가능
- 로컬 파일과 앱을 능동적으로 제어하는 Claude Cowork의 자율성을 외부 오픈소스 모델로 자유롭게 활용

> 포인트: AI 경쟁의 초점이 '개별 모델 성능'에서 '사용자 작업 환경 플랫폼 선점'으로 이동 중

---

### Unity AI 오픈 베타 시작

**Unity 6 이상** 버전을 대상으로 Unity AI 오픈 베타가 시작되었다.

**주요 기능:**
- Unity 워크플로우에 맞춰진 **내장 에이전트** 활용
- **AI Gateway** 및 **MCP Server**를 통해 외부 AI 도구를 엔진에 직접 연결 가능
- 사용자 데이터를 AI 모델 학습에 기본적으로 사용하지 않는다고 명확히 선언 (저작권 문제 대응)
- 프로 구독자 기본 제공, 개인 사용자 **월 10달러**

> 커뮤니티 반응: 저품질 양산형 게임 우려 vs. 1인 개발 한계 극복 기대 엇갈림

## 4. 개발자 도구 & 오픈소스

### Lazyweb — 25만 화면 디자인 레퍼런스 MCP 무료 공개

듀오링고 출신 개발자가 개발한 **Lazyweb**이 완전 무료로 공개되었다.

- 25만 개 이상 실제 앱·웹 화면 데이터 보유
- Claude, Codex 등 AI 모델에 **MCP로 즉시 연동** — 유료 구독·사용량 제한 없음
- AI가 생성한 코드의 엉성한 디자인을 고품질 레퍼런스로 보완

---

### AgentPulse — 애플워치로 Codex 제어

한 개발자가 **애플워치에서 OpenAI Codex 세션을 제어**할 수 있는 오픈소스 프로젝트 **AgentPulse**를 공개했다.

- 데스크톱 Codex 앱 서버 기반으로 작동
- 손목에서 스레드 확인, AI 모델 선택, 작업 답변 즉각 전송 가능
- 이동 중 웨어러블 기기로 AI 에이전트를 원격 조종하는 새로운 작업 환경 구현

---

### YouMind — 실시간 트렌딩 프롬프트 추적기

**YouMind** 웹사이트에서 X(트위터)에서 화제가 되는 프롬프트를 매일 추적해 실시간으로 업데이트하는 기능이 공개되었다.

- 기존 정적 프롬프트 템플릿 모음집과 달리 **검증된 실전 프롬프트 실시간 공유**
- 모델 버전이 빠르게 바뀌는 환경에서 커뮤니티 기반의 최신 프롬프트 참고 가능

---

### Google Pomelli Catalog — 소상공인 무료 AI 마케팅

구글이 소규모 비즈니스 맞춤형 도구 **Pomelli**에 **Catalog** 기능을 추가했다.

- 제품·서비스 정보만 입력하면 AI가 브랜드 특성에 맞춰 **개인화된 마케팅 캠페인 + 고품질 화보** 자동 생성
- **전 세계 어디서나 완전 무료** 제공 → 소상공인 마케팅 진입 장벽 대폭 낮춤
