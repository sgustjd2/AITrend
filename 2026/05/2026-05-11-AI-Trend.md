## 1. 모델 출시 & 업데이트

### OpenAI Deployment Company 출범

OpenAI가 **Forward Deployed Engineer(FDE)**를 기업에 직접 파견하는 새로운 서비스 조직을 출범시켰다. Tomoro 인수로 150명의 현장 엔지니어 조직을 확보했다.

**핵심 특징:**

| 항목 | 내용 |
|------|------|
| 서비스 | FDE(Forward Deployed Engineer) 기업 내부 직접 파견 |
| 목표 | 업무 프로세스·조직 구조를 AI 중심으로 재설계 |
| 참여 기업 | McKinsey, Bain, Goldman Sachs 등 대형 기업 |
| 인력 | Tomoro 인수로 150명 현장 엔지니어 확보 |

> 포인트: OpenAI가 API 제공을 넘어 **컨설팅 + 구현 + 운영을 번들로 파는** 신사업으로 진입했다. AI 전환을 직접 끌어주는 모델이라 전통 컨설팅 회사와 정면 경쟁 구도가 만들어진다.

## 4. 경량 VLM 업데이트 — RTX 3070/4070 구동 가능 모델

> 용도: 이미지를 보고 텍스트로 체크하는 Vision Q&A  
> 예시: "현재 이미지에 화재나 연기가 있나요?", "작업자가 안전고리를 착용하고 있나요?"

### 추천 경량 VLM 모델 목록

| 모델명 | 파라미터 | 예상 VRAM | RTX 3070/4070 | 특징 |
|--------|---------|---------|--------------|------|
| **Qwen3-VL-2B-Instruct** | 2B | ~4GB | ✅ 가능 | 256K 컨텍스트, 32개 언어 OCR, 공간 인식 |
| **Qwen3-VL-4B-Instruct** | 4B | ~6GB | ✅ 가능 | 2B 대비 향상된 추론 성능 |
| **Qwen3-VL-8B-Instruct** | 8B | ~10GB | ✅ 가능 (4070) | 고급 멀티모달 추론, 영상 이해 |
| **MiniCPM-V-4.6** | 1B | ~2GB | ✅ 가능 | 초경량, 엣지 배포 최적화 |
| **SmolVLM-2B** | 2B | ~4GB | ✅ 가능 | HuggingFace 공식, 빠른 추론 |
| **ForestFireVLM-7B** | 7B | ~9GB | ✅ 가능 (4070) | 산불·화재 감지 특화 파인튜닝 |

### Qwen3-VL-2B — 산업 안전 활용 예시

Qwen3-VL-2B는 객체 위치 판단, 폐색(occlusion) 인식, 장면 이해 능력을 갖춰 **산업 안전 모니터링**에 활용 가능하다.

```python
from transformers import Qwen3VLForConditionalGeneration
import torch

model = Qwen3VLForConditionalGeneration.from_pretrained(
    "Qwen/Qwen3-VL-2B-Instruct",
    torch_dtype=torch.bfloat16,
    attn_implementation="flash_attention_2",
    device_map="auto",
)

# 활용 예시 질의
questions = [
    "현재 이미지에 화재나 연기가 보이나요?",
    "작업자가 안전모와 안전고리를 착용하고 있나요?",
    "이 이미지에서 위험 요소가 있다면 설명해주세요.",
]
```

### 화재 감지 특화 모델

| 모델명 | 링크 | 특징 |
|--------|------|------|
| **ForestFireVLM-7B** | [HuggingFace](https://huggingface.co/leon-se/ForestFireVLM-7B) | VLM 기반 산불·화재 설명 생성 |
| **Fire-Detection-Engine** | [HuggingFace](https://huggingface.co/prithivMLmods/Fire-Detection-Engine) | Trainer API 파인튜닝, 높은 F1 score |
| **nanoVLM** | [HuggingFace](https://huggingface.co/blog/nanovlm) | 순수 PyTorch, 커스텀 학습 최적화 |

> 포인트: RTX 3070(8GB VRAM)은 2B~4B 모델을 BF16으로 구동 가능하고, RTX 4070(12GB VRAM)은 7B~8B 모델까지 커버된다. 화재·안전장비 확인 특화 사용이라면 **Qwen3-VL-2B + 파인튜닝** 또는 **ForestFireVLM-7B** 조합이 실용적이다.
