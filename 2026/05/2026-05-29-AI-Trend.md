### 빠른 실행 — Qwen3-VL-2B (화재·안전고리 시각 판단)

```python
from transformers import Qwen2VLForConditionalGeneration, AutoProcessor
from PIL import Image
import torch

# 2B 모델 — RTX 3070 VRAM 3GB
model = Qwen2VLForConditionalGeneration.from_pretrained(
    "Qwen/Qwen3-VL-2B-Instruct",
    torch_dtype=torch.float16,
    device_map="auto"
)
processor = AutoProcessor.from_pretrained("Qwen/Qwen3-VL-2B-Instruct")

image = Image.open("cctv_frame.jpg")
messages = [
    {
        "role": "user",
        "content": [
            {"type": "image", "image": image},
            {"type": "text", "text": "다음 항목을 각각 예/아니오로 답하세요:\n1. 화재 또는 연기가 보입니까?\n2. 작업자가 안전고리(하네스)를 착용하고 있습니까?\n3. 안전모를 착용하고 있습니까?"}
        ]
    }
]

text = processor.apply_chat_template(messages, tokenize=False, add_generation_prompt=True)
inputs = processor(text=[text], images=[image], return_tensors="pt").to(model.device)
outputs = model.generate(**inputs, max_new_tokens=150)
print(processor.decode(outputs[0][inputs["input_ids"].shape[-1]:], skip_special_tokens=True))
```

| **Chartographer** | — | 3 | VLM 평가용 반사실적 차트 생성 |

### 주목 — Gamma-World (NVIDIA, 154표 압도적 1위)

**Gamma-World**는 기존 2인 플레이어 게임 프레임을 넘어 다중 에이전트 환경에서 세계 모델을 학습하는 연구다.

> 포인트: 다중 에이전트 시뮬레이션 환경은 산업 현장 시뮬레이션(여러 작업자의 상호작용, 사고 위험 예측)에 직접 활용 가능한 기반 기술이다. NVIDIA가 게임 기반 연구에서 산업 AI로 전환하는 흐름의 일환.

### 주목 — From Pixels to Words (53표, VLM 핵심)

**From Pixels to Words**는 토큰화 없이 픽셀을 직접 처리해 텍스트를 생성하는 경량 VLM의 새 스케일링 접근법이다.

> 포인트: 기존 VLM은 이미지를 패치 토큰으로 변환하는 별도 단계가 필요하다. 이 논문의 방식은 이를 제거해 모델을 더 단순하고 경량화할 수 있다. 화재·안전 감지처럼 단일 이미지를 빠르게 분류하는 용도에 적합한 미래 아키텍처 방향.

### 주목 — Learn from Weaknesses (KAIST AI, 32표)

**KAIST AI의 Learn from Weaknesses**는 소형 컴퓨터 사용 에이전트가 자신의 약점을 분석해 자동으로 도메인을 특화하는 연구다.

> 포인트: 한국 KAIST가 HF Daily Papers 상위권에 위치한 연구. 소형 에이전트가 특정 도메인(예: 안전 점검)에 자동으로 특화되는 기술은 파인튜닝 비용 없이 경량 모델을 현장에 맞게 조정하는 미래를 시사한다.
