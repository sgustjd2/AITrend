## 1. 모델 & API 동향

### GPT-5.5 Instant — 전체 사용자 롤아웃

OpenAI가 **GPT-5.5 Instant**를 모든 ChatGPT 사용자에게 순차 배포했다. API에서는 `chat-latest` 별칭이 기존 GPT-5.3 Instant에서 GPT-5.5 Instant로 교체됐다.

**핵심 변경:**
- 기존 GPT-5.3 Instant 대비 응답 속도·정확도 향상
- API 호출 시 `chat-latest` 모델명으로 자동 업그레이드 적용
- 무료 플랜 포함 전 티어 대상

> 포인트: `chat-latest` 자동 업그레이드는 하위 호환성 리스크가 있다. 프로덕션 애플리케이션은 명시적 버전 고정이 필요

---

### Google Gemini 3.1 Flash-Lite — 초고속·초저가 효율 모델

Google이 **Gemini 3.1 Flash-Lite**를 출시했다. 기존 Gemini 모델 대비 응답 속도와 비용 효율에 집중한 라인업이다.

**주요 스펙:**

| 항목 | 수치 |
|------|------|
| 응답 속도 | 기존 대비 **2.5배** 향상 |
| 출력 생성 속도 | 기존 대비 **45%** 빠름 |
| 입력 가격 | **$0.25/1M 토큰** |

> 포인트: 대용량 처리·실시간 서비스에 최적화된 포지셔닝. GPT-4o mini와 Claude Haiku를 직접 겨냥한 가격대

## 2. 로보틱스 AI

### Figure Helix-02 — 중앙 제어 없는 2대 협업 자율 정리

**Figure**가 차세대 휴머노이드 **Helix-02** 두 대가 침실을 2분 내에 정리하는 영상을 공개했다.

**시연 내용:**
- 문 열기, 옷 걸기, 쓰레기 버리기, 의자 정리, 침대 정돈 수행
- **중앙 제어 서버 없음** — 로봇 간 직접 메시지 교환 없이 완전 자율 작동
- 각 로봇이 환경 인식 후 독립적으로 역할 분담 결정

> 포인트: '중앙 오케스트레이터 없는 멀티 에이전트 협업'은 로보틱스의 새로운 기준점. 물류·제조·가사 자동화 현장 적용 가속화 신호

## 3. 개발자 도구

### OpenAI Codex — Remote Control (SSH 원격 인스턴스 관리) 추가

OpenAI가 **Codex**에 **Remote Control** 기능을 개발 중임을 확인했다. SSH를 통해 원격 인스턴스를 직접 관리할 수 있는 기능이 추가될 예정이다.

**추가 사항:**
- SSH 원격 인스턴스 연결 토글 설정
- 원격 연결 유지 기능
- Codex 에이전트의 외부 서버 직접 조작 지원 예정

> 포인트: Codex가 클라우드 환경 내 자율 작업 범위를 서버 레벨까지 확장하는 방향. 실제 인프라 조작 에이전트로 진화 중

---

### Coder Agents — 자체 호스팅 AI 개발 워크플로우 플랫폼 베타 출시

AI 개발 인프라 기업 **Coder**가 **Coder Agents**를 베타 공개했다.

**주요 특징:**
- 기업 자체 인프라 위에서 AI 개발 워크플로우 실행
- 원하는 AI 모델 자유 선택 가능 (벤더 종속 없음)
- 에이전트 기반 코드 생성·리뷰·배포 파이프라인 지원

## 4. 보안 & 규제

### 백악관 AI 모델 사전 검토 행정명령 초안 — FDA 방식 도입 추진

백악관 국가경제위원회(NEC) 위원장 Kevin Hassett이 **AI 모델을 공개 전 사전 심사하는 행정명령 초안**을 준비 중이라고 밝혔다.

**배경:**
- Anthropic의 **Mythos 모델**이 네트워크 취약점 자동 탐색 능력을 보유한 것이 촉발
- FDA 의약품 승인 방식을 AI 모델에 적용하는 구조 검토 중

**AI 감독 체계 확대:**
- 미국 AI 표준·혁신센터(CAISI)가 Google DeepMind, Microsoft, xAI와 모델 평가 협약 체결
- 2024년 OpenAI·Anthropic 협약에 이어 빅테크 전반으로 확대

> 포인트: AI 규제가 '사후 관리'에서 '사전 승인' 구조로 전환되는 분기점. 개발 사이클 및 출시 일정에 직접 영향을 줄 수 있어 업계 주목

---

### Snyk-Claude 통합 — AI 코드 보안 플랫폼

Anthropic의 **Claude** 모델이 **Snyk AI Security Platform**에 통합됐다.

- 취약점 자동 식별 및 수정 코드 제안
- 공동 고객사 대상 즉시 사용 가능

## 5. 산업 & 비즈니스 동향

### Anthropic — Q1 연 환산 매출 $44B, 분기 만에 80배 성장

AI Weekly 보도에 따르면 Anthropic의 Q1 수익이 분기 기준 **연 환산 $44B ARR**에 도달했다. 이는 직전 분기 대비 **80배 증가**로 프런티어 AI 기업 최대 분기 성장률이다.

**동반 발표:**
- Google Cloud에 **$200B(약 276조원)** 5년 지출 약정
- SpaceX 컴퓨트 계약 체결
- **Claude Code Auto Mode** 출시
- Jamie Dimon(JPMorgan CEO)과 금융 서비스 에이전트 10개 공동 런칭

---

### Google DeepMind UK — 직원 98% 노동조합 결성 투표

Google DeepMind 영국 직원들이 **98% 찬성으로 노동조합 결성**을 결의했다. 주요 AI 연구소 최초의 노조 결성 사례다.

- 배경: 기밀 Pentagon AI 계약에 대한 투명성 요구
- 연구자·엔지니어 다수가 군사용 AI 활용에 우려 표명

---

### EU AI Act — 준수 기한 2026년 8월 → 2027년 12월 연장

유럽연합이 **EU AI Act** 이행 기한을 기존 2026년 8월에서 **2027년 12월**로 16개월 연장했다.

- 기업들의 준비 기간 부족 우려 반영
- 고위험 AI 시스템 규제 요건 충족 시간 확보
