## 2. AI 인사이트 — 리더들의 발언

### Jensen Huang (NVIDIA CEO) — Stanford CS153

> "생성형 AI의 등장으로 컴퓨터 과학과 모든 과학 분야의 근본이 완전히 바뀌었다."

| 발언 포인트 | 내용 |
|------------|------|
| AI 진화 단계 | 이미지 생성·텍스트 요약 → 스스로 단계를 밟아 추론하고 **생각을 생성(generate thoughts)** 하는 단계 |
| 컴퓨팅 패러다임 | 온디맨드(on-demand) → **지속적으로 실행(continuously running)** 방식 전환 예고 |
| 인프라 재정의 | PC·클라우드·인프라 개념 모두 재정립 필요 |

### Yann LeCun (Meta Chief AI Officer) — 두 가지 발언

**① 진정한 에이전트의 조건**

> "자신의 행동 결과를 예측하는 능력이 없는 에이전트 시스템은 상상할 수 없다."

- 현재 VLA(시각-언어-행동) 모델과 LLM은 인과관계 학습이 불완전해 미래 결과 예측 불가
- 예측 능력 없이는 행동을 지른 후 결과를 수동적으로 기다릴 수밖에 없다
- 안전한 보호 범위(safety guardrails) 내에서 기술 통제하려면 예측 능력이 필수

**② 1~2년 내 계층적 월드 모델 완성**

> "우리는 '월드 모델'을 학습시킬 수 있다는 것을 곧 증명해 보일 것입니다."

| 항목 | 내용 |
|------|------|
| 기술 | 계층적 월드 모델(Hierarchical World Models) |
| 완성 시기 | 1년~18개월 내 |
| 적용 분야 | 로봇공학, 산업 프로세스 제어, 헬스케어 |

### 빠른 실행 — moondream2 (이미지 Q&A 특화)

```python
from transformers import AutoModelForCausalLM, AutoTokenizer
from PIL import Image
import torch

model = AutoModelForCausalLM.from_pretrained(
    "vikhyatk/moondream2",
    trust_remote_code=True,
    torch_dtype=torch.float16,
    device_map="auto"
)
tokenizer = AutoTokenizer.from_pretrained("vikhyatk/moondream2")

image = Image.open("작업현장.jpg")
enc_image = model.encode_image(image)

# 안전 체크 예시
questions = [
    "현재 이미지에 화재나 연기가 보이나요?",
    "작업자가 안전고리(하네스)를 착용하고 있나요?",
    "위험 요소가 있다면 구체적으로 설명해주세요."
]
for q in questions:
    print(f"Q: {q}")
    print(f"A: {model.answer_question(enc_image, q, tokenizer)}\n")
```
