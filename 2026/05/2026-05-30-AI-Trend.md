|------|------|

### 빠른 실행 — LFM2.5-8B-A1B (에이전트형 화재·안전 판단)

```python
from transformers import AutoModelForCausalLM, AutoProcessor
from PIL import Image
import torch

# 8B MoE 모델 — 추론 시 1.5B 활성화, RTX 3070 VRAM ~3-4GB
model = AutoModelForCausalLM.from_pretrained(
    "LiquidAI/LFM2.5-8B-A1B",
    torch_dtype=torch.float16,
    device_map="auto"
)
processor = AutoProcessor.from_pretrained("LiquidAI/LFM2.5-8B-A1B")

image = Image.open("cctv_frame.jpg")
messages = [
    {
        "role": "user",
        "content": [
            {"type": "image", "image": image},
            {"type": "text", "text": (
                "다음 항목을 각각 예/아니오로 답하고 근거를 한 문장으로 설명하세요:\n"
                "1. 화재 또는 연기가 보입니까?\n"
                "2. 작업자가 안전고리(하네스)를 착용하고 있습니까?\n"
                "3. 안전모를 착용하고 있습니까?\n"
                "4. 기타 즉각적인 위험 요소가 있습니까?"
            )}
        ]
    }
]

inputs = processor.apply_chat_template(
    messages, tokenize=True, add_generation_prompt=True,
    return_dict=True, return_tensors="pt"
)
outputs = model.generate(**inputs.to(model.device), max_new_tokens=300)
print(processor.decode(outputs[0], skip_special_tokens=True))
```

### 주목 — Qwen-VLA (Qwen/Alibaba, 107표 2위)

> 포인트: VLM에서 VLA로의 진화는 "보고 말하는" 수준을 넘어 "보고 행동하는" 에이전트로의 전환이다. 산업 현장 CCTV가 단순 이미지 분석을 넘어 경보·제어 시스템으로 연결되는 아키텍처의 기술적 기반이다.

### 주목 — Why Far Looks Up (NVIDIA, 38표)

> 포인트: 안전고리 착용 여부나 작업자의 위치·자세를 정확히 판단하려면 VLM의 공간 추론 능력이 핵심이다. NVIDIA가 VLM의 공간 이해 한계를 정량화한 이 연구는 안전 감지 시스템 정확도 향상의 방향을 제시한다.
