## 3. 경량 VLM (Vision Language Model) 업데이트

> 용도: 이미지 분석 (화재/연기 감지, 작업자 안전고리 착용 여부 확인 등)  
> 타겟 GPU: RTX 3070 (8GB VRAM) / RTX 4070 (12GB VRAM)

---

### ✅ Liquid AI LFM2.5-VL-450M — 신규 주목 모델 (엣지 배포 특화)

| 항목 | 내용 |
|------|------|
| 파라미터 | **450M** (초경량) |
| 추론 속도 | **250ms 이하** (엣지 하드웨어 기준) |
| RTX 3070 | ✅ (매우 여유롭게 실행) |
| RTX 4070 | ✅ |
| 특징 기능 | 바운딩 박스 예측, 다국어 지원, 함수 호출 |
| 공개 일자 | 2026-04-11 |

**핵심 특징**:
- 단일 패스로 **공간적 출력(바운딩 박스) + 의미 추론** 동시 수행
- 개인정보 보호 필요 환경에서 영상을 외부 서버 전송 없이 **온디바이스 처리**
- NVIDIA Jetson Orin 등 임베디드 하드웨어 공식 지원
- 산업 응용: 창고 자동화, 차량 내 인식, 리테일 시각 검색

**안전 점검 활용 예시**:
```python
# 450M 초경량으로 RTX 3070에서 실시간 처리 가능
from liquid import LFMVisionModel

model = LFMVisionModel.from_pretrained("liquid-ai/lfm2.5-vl-450m")

result = model.analyze(
    image="cctv_frame.jpg",
    query="현재 작업자는 안전고리를 착용하고 있나요? 미착용 시 위치를 표시해주세요."
)
# 바운딩 박스 좌표 + 텍스트 분석 결과 동시 반환
```

## 4. RTX 3070/4070 기준 VLM 권장 순위 (2026-06-06 기준)

| 순위 | 모델 | VRAM | 정확도 | 속도 | 추천 용도 |
|------|------|------|--------|------|----------|
| 1 | **Qwen3-VL-4B-Instruct** | ~4GB (4bit) | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 안전 점검 범용 |
| 2 | **Qwen3-VL-8B-Instruct (4bit)** | ~6-7GB | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | RTX 4070 고정밀 분석 |
| 3 | **LFM2.5-VL-450M** | ~1GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 실시간 엣지 배포 (신규) |
| 4 | **InternVL3-2B** | ~2.5GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 초경량 엣지 배포 |
| 5 | **SmolVLM2-2.2B** | ~5GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 실시간 CCTV 스트림 |
| 6 | **YOLO26 + Qwen3-VL-4B** | ~5GB | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 실시간 감시 특화 |

> 💡 **신규 추가**: LFM2.5-VL-450M이 VRAM 1GB 내외로 작동하면서 바운딩 박스까지 출력 — 매우 제한된 환경(임베디드, 다중 카메라)에서 유용. 단, 정확도는 Qwen3-VL 4B 대비 낮음
>
> 💡 **안전 점검 2단계 파이프라인 권장** (기존 유지):
> - **1단계**: YOLO26으로 화재/연기/작업자 실시간 탐지 (ms 단위)
> - **2단계**: 관심 영역(ROI) 크롭 후 Qwen3-VL-4B로 상세 텍스트 분석
> - RTX 3070: YOLO26 + Qwen3-VL-4B 조합 (총 VRAM ~5GB)
> - RTX 4070: YOLO26 + Qwen3-VL-8B (4bit) 조합으로 더 높은 정밀도 확보
