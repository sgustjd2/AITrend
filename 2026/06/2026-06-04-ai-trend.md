## 2. 경량 VLM (Vision Language Model) 업데이트

> 용도: 이미지 분석 (화재/연기 감지, 작업자 안전고리 착용 여부 확인 등)  
> 타겟 GPU: RTX 3070 (8GB VRAM) / RTX 4070 (12GB VRAM)

---

### ✅ Qwen3-VL — Alibaba 경량 멀티모달 (현재 최우선 추천)

| 모델 | 파라미터 | VRAM (4bit) | RTX 3070 | RTX 4070 | 비고 |
|------|---------|-------------|----------|----------|------|
| Qwen3-VL-2B-Instruct | 2B | ~2.5GB | ✅ | ✅ | 초경량 배포용 |
| Qwen3-VL-4B-Instruct | 4B | ~4GB | ✅ | ✅ | 범용 추천 |
| Qwen3-VL-8B-Instruct | 8B | ~6-7GB | ⚠️ 4bit 필요 | ✅ | 고정밀 분석 |
| Qwen3-VL-8B-Thinking | 8B | ~6-7GB | ⚠️ 4bit 필요 | ✅ | 추론 강화 |

**핵심 특징**:
- **256K 컨텍스트** (최대 1M 확장)
- **32개 언어 OCR**, 시각 추론, GUI 자동화, 공간 인식
- **Interleaved-MRoPE** + **DeepStack** 아키텍처 — 멀티레벨 특징 융합
- vLLM, Ollama, LM Studio 모두 지원
- Ollama: `ollama pull qwen3-vl:2b` 즉시 사용 가능

**안전 점검 활용 예시**:
```python
from transformers import Qwen3VLForConditionalGeneration, AutoProcessor

model = Qwen3VLForConditionalGeneration.from_pretrained(
    "Qwen/Qwen3-VL-4B-Instruct",
    torch_dtype="auto",
    load_in_4bit=True  # RTX 3070 권장
)
processor = AutoProcessor.from_pretrained("Qwen/Qwen3-VL-4B-Instruct")

messages = [{
    "role": "user",
    "content": [
        {"type": "image", "image": "image.jpg"},
        {"type": "text", "text": "현재 이미지에 화재나 연기가 있나요? 있다면 위치와 심각도를 알려주세요."}
    ]
}]
# 또는: "현재 작업자는 안전고리(안전벨트)를 착용하고 있나요?"
```

### ✅ VLM + 객체 탐지 결합 연구 — 산업 안전 특화 파이프라인

- **논문**: "Integration of Object Detection and Small VLMs for Construction Safety Hazard Identification" (2026-04)
  - Gemma-3 4B, Qwen3-VL 2B/4B, InternVL3 1B/2B, SmolVLM2-2B 성능 비교
  - 탐지 가이드 sVLM 프레임워크: 객체 탐지 + 멀티모달 추론 통합
  - 오버헤드: **추론당 +2.5ms** (실시간 처리 가능 수준)
  - 건설현장 안전모·안전고리·낙하 위험 탐지 실증

- **논문**: "Towards efficient context-aware classification with compact VLM architectures: indoor fire case study" (Scientific Reports, 2026)
  - 경량 VLM으로 화재 3단계 분류: 화재 없음 / 통제 화재 / 위험 화재
  - 시각 인코더 + VLM 캡션 생성 + 언어 모델 분류 파이프라인
  - RTX 3070급 GPU에서 실시간 처리 가능 확인

**핵심 인사이트**: 단순 VLM 단독 추론보다 **YOLO계 탐지 모델 선행 + VLM 상세 분석** 2단계 파이프라인이 속도와 정확도 모두 우수

## 3. RTX 3070/4070 기준 VLM 최종 권장 순위 (2026-06-04 기준)

| 순위 | 모델 | VRAM | 정확도 | 속도 | 추천 용도 |
|------|------|------|--------|------|----------|
| 1 | **Qwen3-VL-4B-Instruct** | ~4GB (4bit) | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 안전 점검 범용 |
| 2 | **Qwen3-VL-8B-Instruct (4bit)** | ~6-7GB | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | RTX 4070 고정밀 분석 |
| 3 | **InternVL3-2B** | ~2.5GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 초경량 엣지 배포 |
| 4 | **SmolVLM2-2.2B** | ~5GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 실시간 CCTV 스트림 |
| 5 | **YOLO26 + Qwen3-VL-4B** | ~4GB | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 실시간 감시 특화 |

> 💡 **안전 점검 2단계 파이프라인 권장**:
> - **1단계**: YOLO26으로 화재/연기/작업자 실시간 탐지 (ms 단위 고속)
> - **2단계**: 관심 영역(ROI) 크롭 후 Qwen3-VL-4B로 상세 텍스트 분석
> - RTX 3070: YOLO26 + Qwen3-VL-4B 조합 (총 VRAM ~5GB)
> - RTX 4070: YOLO26 + Qwen3-VL-8B (4bit) 조합으로 더 높은 정밀도 확보
> - **Cosmos 3 Edge (4B) 출시 시** RTX 4070 기준 2단계 대체 후보로 재평가 예정
