## 2. HuggingFace 신규 VLM 업데이트 (경량 모델 — RTX 3070/4070 대응)

> 용도: 이미지를 보고 텍스트 질의에 응답하는 산업 안전 체크 (화재/연기 감지, 안전장비 착용 여부 등)

### 추천 모델 목록

#### Fire-Detection-Siglip2 ⭐ 추천
- **모델**: `prithivMLmods/Fire-Detection-Siglip2`
- **아키텍처**: SigLIP2-base-patch16-224 기반 이미지 분류
- **파라미터**: **92.9M** (초경량)
- **분류 클래스**: `화재(Fire)` / `연기(Smoke)` / `정상(Normal)` 3가지
- **성능**: 정확도 **99.41%**, 정밀도/재현율 모두 0.99 이상
- **GPU 적합성**: RTX 3070(8GB)/4070(12GB) 모두 충분한 VRAM → 실행 가능
- **라이선스**: Apache 2.0
- **사용 예시**: `"현재 이미지는 화재나 연기가 있는 건가요?"`
- 설치: `transformers`, `torch` 필요; `AutoImageProcessor + SiglipForImageClassification`으로 추론

```python
from transformers import AutoImageProcessor, SiglipForImageClassification
from PIL import Image
import torch

model_id = "prithivMLmods/Fire-Detection-Siglip2"
processor = AutoImageProcessor.from_pretrained(model_id)
model = SiglipForImageClassification.from_pretrained(model_id)

image = Image.open("your_image.jpg")
inputs = processor(images=image, return_tensors="pt")
with torch.no_grad():
    outputs = model(**inputs)
pred = outputs.logits.argmax(-1).item()
print(model.config.id2label[pred])  # "Fire", "Smoke", or "Normal"
```

#### YOLOv26 Fire Detection
- **모델**: `SalahALHaismawi/yolov26-fire-detection`
- **아키텍처**: YOLO v26 기반 객체 탐지
- **용도**: 이미지 내 화재/연기 **위치(bbox) 탐지** (분류보다 정밀)
- **GPU 적합성**: RTX 3070/4070 실행 가능 (YOLO 계열은 경량 특화)
- **사용 예시**: 화재 발생 위치를 바운딩 박스로 시각화

---

#### Qwen3-VL (2B / 4B 버전) — 범용 VQA
- **모델**: `Qwen/Qwen3-VL-2B-Instruct`, `Qwen/Qwen3-VL-4B-Instruct`
- **아키텍처**: SigLIP2-Large 비전 인코더 + Qwen3 LLM
- **파라미터**: 2B / 4B
- **GPU 적합성**:
  - 2B: RTX 3070(8GB) — 4비트 양자화 시 약 2~3GB VRAM
  - 4B: RTX 4070(12GB) — 4비트 양자화 시 약 4~6GB VRAM
- **장점**: 단순 분류 이상의 **자연어 질의응답** 가능
  - `"현재 작업자는 안전고리를 착용하고 있나요?"` 같은 복잡한 질문 처리 가능
- **추론 권장**: vLLM(>=0.11.0) 또는 transformers

---

#### 기타 경량 VLM
| 모델 | 파라미터 | 특징 |
|------|--------|------|
| `SmolVLM-256M` | 256M | 초경량, 캡셔닝·문서Q&A·기본 추론 |
| `SmolVLM-500M` | 500M | 256M 대비 성능 향상, 여전히 경량 |
| `nanoVLM` (HuggingFace) | ~300M | 프로덕션 배포 최적화, Apache 2.0 |

> **선택 가이드**: 단순 화재/안전장비 분류 → Fire-Detection-Siglip2 / YOLOv26. 자연어 대화형 체크 → Qwen3-VL 2B~4B (양자화 필수).
