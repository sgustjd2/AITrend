### ⚡ TurboVec — 임베딩 데이터 31GB → 4GB 압축, 정확도 유지

- 구글 TurboQuant 논문 기반 오픈소스 라이브러리 TurboVec 공개
  - **31GB 임베딩 데이터를 4GB로 압축**하면서도 검색 정확도 유지
  - 일반 노트북에서 수백만 문서 벡터 검색이 가능해짐
  - 반도체 수요 감소 우려는 과장 — AI 에이전트의 토큰 소비 증가로 총 메모리 수요는 계속 증가 전망
- 💡 **주목 포인트**: RAG 파이프라인 운영 비용을 획기적으로 낮출 수 있는 기술 — 엣지 디바이스 AI 검색, 개인화 임베딩 DB 구축에 즉시 활용 가능

## 3. 경량 VLM (Vision Language Model) 업데이트

> 용도: 이미지 분석 (화재/연기 감지, 작업자 안전고리 착용 여부 확인 등)  
> 타겟 GPU: RTX 3070 (8GB VRAM) / RTX 4070 (12GB VRAM)

---

### 🆕 Keye-VL — 신규 주목 경량 VLM (2026년 등장)

| 항목 | 내용 |
|------|------|
| 파라미터 | **3B** (경량) |
| RTX 3070 | ✅ (4bit 양자화 시 ~3GB VRAM) |
| RTX 4070 | ✅ |
| 특징 | 세밀한 시각 인식 특화, 공간 이해 강점 |
| 출처 | Open-Source VLM Survey 2026 신규 등록 |

**핵심 특징**:
- 세밀한 물체 인식과 공간 관계 이해에 최적화
- 산업 현장 이미지에서 장비/작업자 인식 정확도 향상
- Qwen3-VL과 동등한 경량 옵션으로 부상 중

---

### 🆕 Penguin-VL 2B/8B — Hugging Face 등록 모델 (2026-03-17)

| 항목 | 내용 |
|------|------|
| 파라미터 | **2B** / 8B |
| RTX 3070 | ✅ (2B), ⚠️ 4bit 필요 (8B) |
| RTX 4070 | ✅ (2B/8B 모두) |
| 특징 | LLM 기반 비전 인코더, 효율성 한계 탐구 |
| 논문 | "Penguin-VL: Exploring the Efficiency Limits of VLM" |

**핵심 특징**:
- LLM 기반 비전 인코더를 사용해 일반 CLIP 인코더 대비 다양한 장면 이해력 향상
- 2B 버전은 RTX 3070에서도 실시간 처리 가능
- 안전 점검 멀티샷 프롬프팅에 적합

---

### ✅ 기존 추천 모델 현황 (지속 유효)

| 모델 | 파라미터 | VRAM (4bit) | RTX 3070 | RTX 4070 | 특징 |
|------|---------|-------------|----------|----------|------|
| Qwen3-VL-4B-Instruct | 4B | ~4GB | ✅ | ✅ | 범용 추천 (1순위) |
| Qwen3-VL-8B-Instruct | 8B | ~6-7GB | ⚠️ 4bit 필요 | ✅ | 고정밀 분석 |
| LFM2.5-VL-450M | 450M | ~1GB | ✅ | ✅ | 초경량 엣지, 바운딩박스 지원 |
| InternVL3-2B | 2B | ~2.5GB | ✅ | ✅ | 초경량 배포 |
| SmolVLM2-2.2B | 2.2B | ~5GB | ✅ | ✅ | 실시간 CCTV 스트림 |

---

### 🔥 화재·안전 점검 특화 모델 현황

| 모델 | 특징 | RTX 3070 | 비고 |
|------|------|----------|------|
| **Qwen2.5-VL-3B** (연기 감지) | 3B 버전이 7B보다 연기 감지 정확도 높음 | ✅ | 제로샷 연기 감지 최적화 |
| **ForestFireVLM-7B** | Qwen 기반 산불·화재 파인튜닝 특화 | ⚠️ 4bit 필요 | 화재 감지 전용 파인튜닝 |
| **YOLO26 + Qwen3-VL-4B** | 실시간 탐지 + 상세 분석 2단계 파이프라인 | ✅ (~5GB) | 현장 CCTV 최적 조합 |

**안전 점검 2단계 파이프라인 (권장)**:
```python
# 1단계: YOLO26으로 화재/연기/작업자 실시간 탐지 (ms 단위)
# 2단계: ROI 크롭 후 Qwen3-VL-4B로 상세 텍스트 분석

# 예시 쿼리
queries = [
    "현재 이미지에 화재나 연기가 있나요?",
    "현재 작업자는 안전고리(안전벨트)를 착용하고 있나요?",
    "작업자가 안전모를 착용했나요? 미착용 시 위치를 표시해주세요."
]
# RTX 3070: YOLO26 + Qwen3-VL-4B (총 VRAM ~5GB)
# RTX 4070: YOLO26 + Qwen3-VL-8B 4bit (더 높은 정밀도)
```

## 4. RTX 3070/4070 기준 VLM 권장 순위 (2026-06-08 기준)

| 순위 | 모델 | VRAM | 정확도 | 속도 | 추천 용도 |
|------|------|------|--------|------|----------|
| 1 | **Qwen3-VL-4B-Instruct** | ~4GB (4bit) | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 안전 점검 범용 |
| 2 | **Qwen3-VL-8B-Instruct (4bit)** | ~6-7GB | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | RTX 4070 고정밀 분석 |
| 3 | **LFM2.5-VL-450M** | ~1GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 실시간 엣지 (다중 카메라) |
| 4 | **Penguin-VL-2B** | ~2.5GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 신규 경량 옵션 |
| 5 | **Keye-VL-3B** | ~3GB | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 세밀한 장면 인식 |
| 6 | **InternVL3-2B** | ~2.5GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 초경량 엣지 배포 |
| 7 | **YOLO26 + Qwen3-VL-4B** | ~5GB | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 실시간 감시 최적 조합 |

> 💡 **신규 추가**: Penguin-VL-2B, Keye-VL-3B — 2026년 등장한 신규 경량 옵션으로 기존 InternVL3-2B와 경쟁 구도 형성  
> 💡 **Qwen2.5-VL-3B**: 연기 감지 특화 — 3B 버전이 7B보다 제로샷 연기 탐지 정확도가 높다는 연구 결과 존재
