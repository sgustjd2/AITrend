### 🆕 Sapiens2 — 인간 중심 초경량 비전 모델 (HuggingFace 신규)

| 항목 | 내용 |
|------|------|
| 출시일 | 2026-06-03 |
| 개발사 | Meta AI Research |
| 파라미터 | **0.4B ~ 5B** (다단계 크기) |
| 학습 데이터 | ~10억 장의 인간 중심 이미지 |
| 특화 | 인체 자세 추정, 신체 부위 분할, 깊이 추정 |
| RTX 3070 | ✅ 0.4B~2B 모델 실시간 실행 가능 |
| RTX 4070 | ✅ 5B까지 원활 |
| 안전 점검 활용 | ✅ 작업자 신체 자세·부위 인식 → 안전고리·안전모 착용 여부 판단 보조 |

**안전 점검 활용 포인트**:
- 작업자 포즈 추정으로 **고소 작업 여부** 및 **안전벨트 착용 자세** 분석
- 신체 분할 마스크로 **안전모 착용 영역** 정밀 탐지

### 🔥 화재·안전 점검 파이프라인 (2026-06-13 기준 최신)

```python
# RTX 3070 최적 구성 (총 VRAM ~5-7GB)
queries = [
    "현재 이미지에 화재나 연기가 있나요?",
    "현재 작업자는 안전고리(안전벨트)를 착용하고 있나요?",
    "작업자가 안전모를 착용했나요? 미착용 작업자 위치를 알려주세요.",
    "화재 위험 요소나 가연성 물질이 노출되어 있나요?"
]

# Step 1: YOLO26-Small 실시간 탐지 (~1-2GB VRAM, ms 단위)
# Step 2: 작업자 ROI → Sapiens2-0.4B 포즈 추정 (~0.8GB)
# Step 3: 크롭 ROI → Qwen3-VL-4B 자연어 분석 (~4GB)
```
