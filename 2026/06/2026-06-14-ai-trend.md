## 3. 경량 VLM (Vision Language Model) 업데이트

> 용도: 이미지 분석 (화재/연기 감지, 작업자 안전고리·안전모 착용 여부 확인 등)  
> 타겟 GPU: RTX 3070 (8GB VRAM) / RTX 4070 (12GB VRAM)

---

### 📌 주목 경량 VLM 현황 (2026-06-13 기준)

| 모델 | 파라미터 | VRAM (4bit) | RTX 3070 | RTX 4070 | 특징 |
|------|---------|-------------|----------|----------|------|
| **SmolVLM2-2B** | 2B | ~5GB | ✅ | ✅ | Apache 2.0, 실시간 스트림 가능 |
| **nanoVLM** | 222M | ~1GB | ✅ | ✅ | HuggingFace 공식, 순수 PyTorch |
| **MiniCPM-V 2.6** | ~8B | ~5.5GB | ⚠️ 4bit 필요 | ✅ | 소형 모델 중 강력한 성능 |
| **Qwen2-VL-7B** | 7B | ~6-7GB | ⚠️ 4bit 필요 | ✅ | Object Detection Grounding 파인튜닝 지원 |
| **G2VLM-2B** (CVPR 2026) | 2B | ~2.5GB | ✅ | ✅ | 3D 재구성 + 공간 추론 특화 |

---

### 🔥 SmolVLM2 — 최적의 경량 실용 VLM

| 항목 | 내용 |
|------|------|
| 개발사 | HuggingFace (공식) |
| 파라미터 | **2B** (SmolLM2 1.7B 기반) |
| 최소 VRAM | **5.02GB** |
| 라이선스 | Apache 2.0 (완전 오픈소스) |
| RTX 3070/4070 | ✅ 8GB/12GB VRAM 모두 충분 |
| 벤치마크 | MMMU 38.8 / DocVQA 81.6 / TextVQA 72.7 |
| 안전 점검 활용 | ✅ 화재·연기 감지, 안전 장비 착용 여부 질의응답 |

**안전 점검 예시 쿼리**:
```python
queries = [
    "현재 이미지에 화재나 연기가 있나요?",
    "현재 작업자는 안전고리(안전벨트)를 착용하고 있나요?",
    "작업자가 안전모를 착용했나요?",
    "화재 위험 요소나 가연성 물질이 노출되어 있나요?"
]
```

| 논문 | CVPR 2026 채택 |

| 파라미터 | **2B** |
| RTX 3070 | ✅ 실시간 가능 |
| 안전 점검 활용 | ✅ 작업자 위치·공간 관계 분석 (고소 작업 판단 등) |

---

### ✅ 화재·안전 감시 파이프라인 권장 구성 (RTX 3070 기준)

```
Step 1: YOLO (실시간 객체 탐지, ~1-2GB VRAM)
    ↓ 작업자/화재 ROI 크롭
Step 2: SmolVLM2-2B 또는 Qwen2-VL-7B-4bit (자연어 쿼리 분석, ~5-7GB VRAM)
    ↓ 결과 텍스트 반환
Step 3: 알림 로직 (임계값 초과 시 경고 발생)
```
