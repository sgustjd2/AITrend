## 2. 경량 VLM (Vision Language Model) 업데이트

> 용도: 이미지 분석 (화재/연기 감지, 작업자 안전고리 착용 여부 확인 등)  
> 타겟 GPU: RTX 3070 (8GB VRAM) / RTX 4070 (12GB VRAM)

---

### ✅ 신규 추천: Qwen3-VL 시리즈 (2026년 최신)

| 모델 | 파라미터 | VRAM (FP16) | VRAM (4bit) | RTX 3070 | RTX 4070 |
|------|---------|-------------|-------------|----------|----------|
| Qwen3-VL-2B | 2B | ~5GB | ~2.5GB | ✅ | ✅ |
| Qwen3-VL-4B | 4B | ~8GB | ~4GB | ✅ | ✅ |
| Qwen3-VL-8B | 8B | ~16GB | ~6-8GB | ⚠️ 4bit 필요 | ✅ |
| Qwen3-VL-32B | 32B | ~64GB | ~18GB | ❌ | ❌ |

**핵심 기술**:
- **DeepStack**: 다계층 ViT 피처 융합으로 세밀한 이미지-텍스트 정렬 강화
- **Interleaved-MRoPE**: 시간·너비·높이 전 주파수 위치 임베딩 (영상 추론 강화)
- **Thinking 에디션** 별도 제공: 추론 강화 버전 (Qwen3-VL-8B-Thinking)

**RTX 3070/4070 추천 선택**:
```bash
# 2B (가장 경량, 3070 완벽 호환)
huggingface-cli download Qwen/Qwen3-VL-2B-Instruct

# 4B (성능/메모리 균형, 두 GPU 모두 호환)
huggingface-cli download Qwen/Qwen3-VL-4B-Instruct

# 8B (4bit 양자화, RTX 4070 권장)
huggingface-cli download Qwen/Qwen3-VL-8B-Instruct
```

**안전 점검 활용 예시**:
```python
from transformers import Qwen2VLForConditionalGeneration, AutoTokenizer
model = Qwen2VLForConditionalGeneration.from_pretrained(
    "Qwen/Qwen3-VL-4B-Instruct", torch_dtype="auto"
)
# 화재/연기 감지
question = "현재 이미지에 화재나 연기가 있나요? 있다면 위치와 심각도를 알려주세요."
# 안전고리 착용 확인
question = "현재 작업자는 안전고리(안전벨트)를 착용하고 있나요? 착용 여부를 확인해주세요."
```

### ✅ SmolVLM2 — HuggingFace 초경량 비디오 지원 VLM

| 항목 | 내용 |
|------|------|
| 모델 크기 | 256M / 500M / 2.2B |
| 비디오 지원 | ✅ (최초 초경량 비디오 VLM) |
| VRAM (2.2B) | ~5GB |
| 라이센스 | Apache 2.0 |

**주요 특징**:
- 256M, 500M은 **역대 최소 크기** 비디오 언어 모델
- 2.2B는 수학 문제, 텍스트 이해, 다이어그램, 과학적 시각 질문 개선
- **메모리 소비 대비 성능** 기준 기존 모델 대비 최고 수준

```bash
huggingface-cli download HuggingFaceTB/SmolVLM2-2.2B-Instruct
```

### 🔍 YOLO 계열 최신 동향 (2026)

| 버전 | 특징 | 용도 | RTX 3070/4070 |
|------|------|------|---------------|
| **YOLO26** | NMS-free 엔드투엔드, 멀티태스크 | 엣지 배포, 저전력 | ✅ |
| **YOLOv13** | HyperGraph 기반, YOLOv12 대비 mAP +1.5% | 정밀 객체 탐지 | ✅ |
| **YOLOv12** | Attention-Centric CNN 속도 유지 | 실시간 탐지 | ✅ |

**YOLO26 핵심 특징** (2026년 1월 출시):
- 완전 엔드투엔드 NMS-free 아키텍처
- 객체 탐지 + 인스턴스 분할 + 분류 + 포즈 추정 + 회전 탐지 **5종 동시 지원**
- 엣지 디바이스 및 저전력 환경 최적화
- 소형 물체 정확도 및 CPU 추론 속도 향상

**안전 감시 시스템 활용 예시**:
```python
from ultralytics import YOLO
model = YOLO("yolo26n.pt")  # nano: 가장 경량

# 실시간 안전 점검
results = model.predict(source="camera.jpg", conf=0.5)
# 탐지 항목: 사람, 안전모, 안전고리, 연기, 화재 등
```

## 3. RTX 3070/4070 기준 VLM 최종 권장 순위 (2026-06-01 기준)

| 순위 | 모델 | VRAM | 정확도 | 속도 | 추천 용도 |
|------|------|------|--------|------|----------|
| 1 | **Qwen3-VL-4B** | ~4GB (4bit) | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 안전 점검 범용 |
| 2 | **SmolVLM2-2.2B** | ~5GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 실시간 처리 |
| 3 | **Qwen3-VL-2B** | ~2.5GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 초경량 배포 |
| 4 | **Qwen3-VL-8B (4bit)** | ~7GB | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | 고정밀 요구 시 |
| 5 | **YOLO26 + LLM** | ~4GB | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 실시간 감시 특화 |

> 💡 **안전 점검 파이프라인 권장**:
> - 1단계: YOLO26으로 화재/연기/사람 실시간 탐지
> - 2단계: 관심 영역(ROI) 크롭 후 Qwen3-VL-4B로 상세 분석
> - 이 조합이 RTX 3070/4070에서 최적 성능/메모리 균형 제공
