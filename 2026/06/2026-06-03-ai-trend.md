### 🔬 Claude Code Dynamic Workflows — Anthropic 엔지니어 실무법 공개

- 핵심 철학: **"문제마다 최적의 일회성 하네스(Harness)를 실시간 조립해 실행"**
- 복잡한 작업을 여러 하위 에이전트로 분해, 병렬 처리, 상호 검증 지원
- **에이전트 3대 실패 모드** 공식 명명:
  1. 에이전트 나태함 (Sycophancy) — 작업 중간 포기
  2. 자기 확증 편향 (Self-confirmation bias) — 버그를 스스로 묵인
  3. 목표 이탈 (Goal drift) — 초기 지시사항을 잃어버리는 현상
- `/fork` 기능 업데이트: 시스템 프롬프트·도구·대화 기록 복사 후 백그라운드 에이전트 실행
  - 기존 `/fork`는 `/branch`로 이름 변경
- 💡 **주목 포인트**: AI 코딩 에이전트 실패 유형을 체계화 — 프롬프트 엔지니어링과 워크플로우 설계의 새로운 기준점

## 2. 경량 VLM (Vision Language Model) 업데이트

> 용도: 이미지 분석 (화재/연기 감지, 작업자 안전고리 착용 여부 확인 등)  
> 타겟 GPU: RTX 3070 (8GB VRAM) / RTX 4070 (12GB VRAM)

---

### ✅ Qwen3.5-VL — Alibaba 네이티브 멀티모달 (NVIDIA 공식 지원 추가)

- **기본 모델**: 397B-A17B (MoE 구조) — 대형 클라우드 전용
- **경량 파생 시리즈** (RTX 3070/4070 대상):

| 모델 | 파라미터 | VRAM (4bit) | RTX 3070 | RTX 4070 |
|------|---------|-------------|----------|----------|
| Qwen3.5-VL-2B | 2B | ~2.5GB | ✅ | ✅ |
| Qwen3.5-VL-4B | 4B | ~4GB | ✅ | ✅ |
| Qwen3.5-VL-9B | 9B | ~6-7GB | ⚠️ 4bit 필요 | ✅ |

**핵심 특징**:
- **네이티브 멀티모달 구조** (텍스트·이미지 동시 학습 — 별도 비전 인코더 없음)
- 256K 토큰 컨텍스트 (최대 1M 확장 가능)
- 200개 이상 언어 지원
- 시각 추론·UI 이해·코딩 복합 작업에 강함
- NVIDIA NeMo AutoModel 및 GPU Accelerated Endpoint 공식 지원

**안전 점검 활용 예시**:
```python
from transformers import Qwen2_5VLForConditionalGeneration, AutoProcessor
model = Qwen2_5VLForConditionalGeneration.from_pretrained(
    "Qwen/Qwen3.5-VL-4B-Instruct",
    torch_dtype="auto",
    load_in_4bit=True  # RTX 3070 권장
)
# 화재/연기 감지
question = "현재 이미지에 화재나 연기가 있나요? 있다면 위치와 심각도를 알려주세요."
# 안전고리 착용 확인
question = "현재 작업자는 안전고리(안전벨트)를 착용하고 있나요? 착용 여부를 상세히 확인해주세요."
```

## 3. RTX 3070/4070 기준 VLM 최종 권장 순위 (2026-06-03 기준)

| 순위 | 모델 | VRAM | 정확도 | 속도 | 추천 용도 |
|------|------|------|--------|------|----------|
| 1 | **Qwen3.5-VL-4B** | ~4GB (4bit) | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 안전 점검 범용 |
| 2 | **InternVL3.5-8B (4bit)** | ~6GB | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | 고정밀 안전 점검 |
| 3 | **SmolVLM2-2.2B** | ~5GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 실시간 처리 |
| 4 | **Qwen3.5-VL-2B** | ~2.5GB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 초경량 배포 |
| 5 | **YOLO26 + Qwen3.5-VL-4B** | ~4GB | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 실시간 감시 특화 |

> 💡 **안전 점검 파이프라인 권장** (변경 없음):
> - 1단계: YOLO26으로 화재/연기/사람 실시간 탐지 (고속)
> - 2단계: 관심 영역(ROI) 크롭 후 Qwen3.5-VL-4B 또는 InternVL3.5-8B로 상세 분석
> - RTX 3070: YOLO26 + Qwen3.5-VL-4B 조합 권장
> - RTX 4070: YOLO26 + InternVL3.5-8B (4bit) 조합으로 더 높은 정밀도 확보 가능
