## 1. 주요 AI 이슈 (@choi.openai Threads 기반)

### Robbyant, 인터랙티브 월드 모델 "LingBot-World 2.0" 오픈소스 공개

- 앤트그룹(Ant Group) 계열사 Robbyant가 인터랙티브 월드 모델 LingBot-World 2.0을 오픈소스로 공개
- 720p·60fps로 1시간 연속 재생해도 화질 저하가 없는 것이 특징 — 기존 구글 Genie는 재생 몇 분 후부터 화질 왜곡이 발생했던 것과 대비됨
- 인과 사전학습(causal pretraining)과 MoBA(Mixture of Block Attention) 어텐션 구조로 누적 오차를 줄이는 방식 채택
- 💡 **주목 포인트**: 로봇 학습용 시뮬레이터 등으로 활용 가능한 장시간 안정 재생 월드 모델 — 중국 빅테크의 월드모델 경쟁이 오픈소스 영역까지 확산되는 흐름

### Meta, 첫 유료 API "Muse Spark 1.1" 출시 — 최저가 전략

- Meta가 자체 모델 API 시장에 처음 진입. Muse Spark 1.1을 입력 $1.25/출력 $4.25 (100만 토큰당) 가격으로 공개, 신규 가입자에게 $20 크레딧 제공
- Claude Opus(입력 $5/출력 $25), GPT-5.5(입력 $5/출력 $30) 대비 압도적으로 저렴한 가격 책정
- 에이전트 성능, 도구 사용, 컴퓨터 조작(computer use) 능력이 강화된 버전으로 소개됨
- 💡 **주목 포인트**: 그동안 무료 오픈모델 배포에 집중하던 Meta가 유료 API 시장에 최저가로 진입 — 가격 경쟁이 한 단계 더 격화될 신호

### OpenAI GPT-Live, 후속 반응 지속 — "음성이 새로운 UI"

- 07-08 리포트에서 다룬 완전 양방향(풀듀플렉스) 음성 모델 GPT-Live에 대한 후속 반응이 하루 뒤에도 이어짐

## 2. HuggingFace 신규 VLM 업데이트 (경량 모델 — RTX 3070/4070 대응)

> 용도: 이미지를 보고 텍스트 질의에 응답하는 산업 안전 체크 (화재/연기 감지, 안전장비 착용 여부 등)

### 확인 결과 — 공식 신규 프론티어 VLM 릴리스 없음 (07-08과 동일)

- `image-text-to-text` 카테고리를 생성일 최신순으로 확인. 상위 노출 항목(belegant-4b, Qwen3.5-0.8B 파인튜닝, Gemma4 변형, SmolVLM2-500M-Video-Instruct 등)은 모두 개인/커뮤니티의 파인튜닝·양자화 버전이며, Qwen·Google·Ultralytics 등에서 낸 공식 신규 베이스 모델은 확인되지 않음
- 유일하게 눈에 띄는 항목은 **SmolVLM2-500M-Video-Instruct** 파인튜닝 — 기존 SmolVLM2 라인업에 비디오 인스트럭션 학습이 추가된 커뮤니티 버전 (0.5B, 초경량)
- YOLO 계열 신규 업로드는 이번 조회에서 노출되지 않음

### 선택 가이드 (누적, 변경 없음)

> - 초경량 단순 이진 질의 → **Qwen3.5-0.8B** 또는 **SmolVLM-256M/500M** (비디오 질의 시 SmolVLM2-500M-Video-Instruct 파인튜닝 고려 가능)
> - 화재/연기 위치 탐지 → **YOLO26** (실시간, 경량)
> - 복합 자연어 체크 → **Qwen3-VL 2B~4B** (양자화 필수), **Ministral-3-3B** 또는 **Gemma 4 E2B/E4B** (RTX 4070 12GB 대응)
> - 위치 기반 탐지 + 자연어 → **VLX-Seek**
>
> 지난 리포트들과 동일하게 이번 주에도 위 조합이 최선의 선택지로 유효함. 신규 프론티어 VLM 릴리스가 나오면 즉시 갱신 예정.
