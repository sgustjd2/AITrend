### MiniCPM-V 4.6 — Ollama 공식 통합 (2026-06-25) ⭐ 신규

- **모델**: `openbmb/MiniCPM-V-4.6`

- **아키텍처**: 혼합 4x/16x 비주얼 토큰 압축
- **GPU 적합성**: RTX 3070(8GB)/4070(12GB) 모두 충분 — 로컬 추론 가능
- **주요 기능**: 단일 이미지, 다중 이미지, 영상 이해 지원
- **주목**: 2026-06-25 Ollama 공식 라이브러리에 병합 → `ollama run minicpm-v:1b`로 즉시 실행

- **용도**: 이미지 내 특정 객체(안전고리, 안전모, 소화기 등) **위치 탐지 + 존재 여부 판별**
- **GPU 적합성**: 온디바이스(엣지) 타깃 설계 → RTX 3070 이상에서 충분히 동작
- 💡 작업자 안전장비 착용 여부를 이미지 내 위치 기반으로 확인하는 용도에 특히 적합

### SmolVLM2 — 256M / 500M / 2.2B 모델 패밀리

| 모델 | 파라미터 | 특징 |
|------|--------|------|
| `SmolVLM-256M` | 256M | 세계 최소 VLM. 캡셔닝·문서Q&A·기본 추론 |
| `SmolVLM-500M` | 500M | 256M 대비 DocVQA/MMMU 점수 현저히 향상 |
| `SmolVLM2-2.2B` | 2.2B | 비디오 이해까지 지원, CLIP 기반 비전 인코더 |

- RTX 3070(8GB)/4070(12GB): 세 모델 모두 VRAM 여유 있게 실행 가능
- 라이선스: Apache 2.0
- 용도: `"이 이미지에서 연기가 보이나요?"` 수준의 단순 VQA에서 문서 OCR까지

---

### YOLO26 — 엣지 퍼스트 Vision AI (2026년 1월 출시, 현재 최신 YOLO)

- **모델**: `Ultralytics/YOLO26`
- **아키텍처**: NMS 제거(엔드투엔드), Distribution Focal Loss 제거(엣지 최적화)
- **성능**: YOLO11 Nano 대비 CPU 추론 **43% 빠름**
- **지원 태스크**: 탐지, 인스턴스 세그멘테이션, 포즈 추정, 분류, 방향성 객체 탐지
- **GPU 적합성**: RTX 3070/4070 — Nano~Small 사이즈로 실시간 추론 가능
- **용도**: 화재/연기 **위치(bounding box) 탐지**, 안전장비 착용 여부 탐지
  - 커스텀 파인튜닝 데이터셋으로 화재·안전장비 특화 모델 훈련 가능

---

### Qwen3-VL (2B / 4B) — 범용 VQA (지속 권장 모델)

- **모델**: `Qwen/Qwen3-VL-2B-Instruct`, `Qwen/Qwen3-VL-4B-Instruct`
- **GPU 적합성**:
  - 2B → RTX 3070(8GB): 4비트 양자화 시 약 2~3GB VRAM
  - 4B → RTX 4070(12GB): 4비트 양자화 시 약 4~6GB VRAM
- **장점**: 단순 분류 이상의 **복합 자연어 질의** 처리
  - `"현재 작업자는 안전고리를 착용하고 있나요?"` 같은 복잡한 질문 가능
  - NVIDIA NVFP4 재압축으로 추론 속도 추가 향상 가능 (위 NVIDIA 이슈 참조)

> **선택 가이드**:
> - 화재/연기 위치 탐지 → **YOLO26** (실시간, 경량)
> - 단순 VQA (있다/없다) → **SmolVLM-256M/500M** 또는 **MiniCPM-V 4.6**
> - 복합 자연어 체크 (`"안전고리 착용 여부"`) → **Qwen3-VL 2B~4B** (양자화 필수)
> - 위치 기반 탐지 + 자연어 → **VLX-Seek**
